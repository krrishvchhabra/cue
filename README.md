Banks lose cross-sell opportunities every day because they wait for customers to ask. Cue flips this model.

Cue is an Agentic AI system for SBI that continuously monitors transaction patterns to detect real-life events - a salary hike as shown in this demo or other cases - and proactively engages the customer with the right financial product at the right moment.

Unlike rule-based systems, Cue uses a signal clustering approach: no single transaction triggers an action. Instead, multiple weak signals are combined to infer a high-confidence life event, reducing false positives and making recommendations feel natural, not intrusive.

The result: an AI that doesn't wait to be asked, it anticipates, engages, and converts.
